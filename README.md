# Portale Rimborsi Fondo Economale CNR-ISP Messina

Portale web per la gestione delle richieste di rimborso tramite fondo economale. I dipendenti registrano le proprie spese, allegano la foto dello scontrino e tengono traccia dello stato del rimborso. La contabilità gestisce la coda dei rimborsi, archivia le transazioni ed esporta i dati in CSV.

## Workflow operativo

### Dipendente
1. Apri il portale e accedi con nome + PIN di accesso.
2. Tab **Registra una spesa** → scegli categoria, inserisci il totale dello scontrino, la data, allega la foto, aggiungi una nota se necessario.
3. Premi **Invia richiesta** — la spesa appare subito in **Le tue spese** con stato *In attesa*.
4. Quando la contabilità agisce, lo stato diventa *Rimborsata* oppure *Rifiutata* con la motivazione.

### Contabilità
1. Apri il portale → tab **Sportello contabilità** → inserisci il PIN sportello.
2. Nella coda **In attesa di rimborso**, apri lo scontrino, poi premi **Segna pagata** o **Rifiuta**.
3. Tutte le voci restano nell'**Archivio**, ricercabile e scaricabile in CSV.
4. Registra la spesa in **U-GOV**: Ciclo Acquisti → Generico di Uscita → Fondo Economale (CORSI_AUT). Il portale tiene traccia delle richieste e dei giustificativi; la registrazione contabile formale resta nel gestionale.

## Riferimenti normativi

- CNR — *Regolamento di Amministrazione, Contabilità e Finanza*, versione novembre 2024, artt. 18–22 (fondi economali e pagamenti in forma diretta).
- D.L. 138/2011, art. 2, comma 4-ter (limite unitario per i pagamenti in contanti).
- Procedura U-GOV *Acquisto via Fondo Economale* (Cineca).
